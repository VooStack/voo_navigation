## [1.0.0] - 2025-01-10

### Added
- Initial release of the voo_navigation package example app
- Basic examples demonstrating core voo_navigation functionality
- Examples include:
  - Basic adaptive navigation setup
  - Navigation item configuration
  - Theme preset usage
  - GoRouter integration examples
  - Per-page customization with VooPage
