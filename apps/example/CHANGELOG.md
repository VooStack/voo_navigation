# Changelog

All notable changes to the voo_navigation_showcase example app will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-10

### Added
- Initial release of the voo_navigation showcase app
- Demonstrates all voo_navigation features and theme presets
- Examples for:
  - Adaptive navigation (automatic switching between bottom nav, rail, drawer)
  - Theme presets (glassmorphism, liquidGlass, blurry, neomorphism, material3Enhanced, minimalModern)
  - GoRouter integration with StatefulShellRoute
  - Per-page scaffold overrides with VooPage
  - Navigation sections and grouping
  - Badge notifications
  - User profile footer
  - Collapsible drawer-to-rail mode
